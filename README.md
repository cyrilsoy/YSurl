## 概述

## 修改

在原来的基础上增加自适应移动端,随机ACG背景

## 安装
YSurl 基于 PHP、SQLite 进行开发，您只需要修改 `config.php` 的相关配置并把 inc 目录权限设置为可读写即可。

对于 `Nginx` 用户，则还需把 `nginx.txt` 里面的内容添加到 Nginx 的配置文件里。

添加伪静态

```
  location / {
    try_files $uri $uri/ =404;
    rewrite (\d+|\w+)$ /index.php?id=$1;

    location ^~ /asset/ {
      root /var/www/crz.im;
    }

    location ^~ /api/ {
      root /var/www/crz.im;
    }

    location ^~ /inc/ {
      return 403;
    }
  }
```
